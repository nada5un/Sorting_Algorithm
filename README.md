# Sorting_Algorithm
정렬 알고리즘 

- 선택 정렬
  
  데이터가 무작위로 여러 개 있을 때 가장 작은 데이터를 선택, 맨 앞에 있는 데이터와 바꿈. 
  
  그 다음 작은 데이터를 선택, 앞에서 두번째 데이터와 바꿈 ... 총 n-1번 반복 
  
  -> 알고리즘 풀이에는 느림 


- 삽입 정렬

  데이터가 거의 정렬 되어 있을 때 효율적.
  
  특정한 데이터를 적절한 위치에 삽입
  

- 퀵 정렬

  데이턱 무작위로 입력되는 경우 효율적.
  
  pivot 존재 
  

- 계수 정렬

  특정 조건이 부합할 때만 사용 가능. 동일한 값을 가지는 데이터가 여러개 등장할 때 적합.
